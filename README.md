# BrowOS

A browser-based operating system with Apple-like aesthetics, built with HTML, CSS, and JavaScript.

## Features

- macOS-inspired user interface
- Window management system
- Dock with magnification effect
- Menu bar with clock
- Multiple applications (FileBrow, Safari, Messages, Settings)
- File system simulation
- Responsive design

## Applications

1. **FileBrow** - File manager
2. **Safari** - Web browser simulation
3. **Messages** - Messaging app simulation
4. **Settings** - System preferences

## Architecture

- `index.html` - Main entry point
- `css/` - Stylesheets for the OS and applications
- `js/` - JavaScript modules for functionality
- `assets/` - Icons and other static assets

## Development

To run BrowOS locally:

1. Clone or download this repository
2. Open `index.html` in a web browser

No build process is required - it's pure client-side JavaScript.

## Customization

You can customize the look and feel by modifying the CSS files in the `css/` directory.
